### 软件介绍

- 一个Qt开发的免费开源的国产录屏软件，将来还会扩展类似于VNC一样的远程控制的功能。

### 软件开发环境（兼容Qt6）
*  以下是作者经常编译的版本
*  C++11
*  QtCreator9.0.2 + Qt6.5.1 + MinGW 64-bit
*  推荐使用Qt6.x进行开发
